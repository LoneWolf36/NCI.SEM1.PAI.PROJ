# Submission Checklist

## Mandatory artefacts

- [x] Project report PDF
- [x] Code artefact zip/gz
- [ ] Presentation mp4
- [x] Journal PDF — member 1
- [x] Journal PDF — member 2
- [x] Journal PDF — member 3
- [x] Journal PDF — member 4

## Report

- [x] IEEE conference format used
- [x] Front page has full names and student numbers
- [x] Abstract included
- [x] Introduction included
- [x] Related Work included
- [x] Methodology included
- [x] Results and Evaluation included
- [x] Conclusions and Future Work included
- [x] Bibliography included
- [x] In-text IEEE citations consistent
- [x] All claims match code evidence
- [x] Novelty, significance, impact stated explicitly

## Data / pipeline compliance

- [x] 3 or 4 datasets used
- [x] Each dataset has at least 1,000 records
- [x] Raw datasets stored programmatically in database before processing
- [x] Cleaned / transformed datasets stored programmatically
- [x] Processed outputs stored in database where relevant
- [x] End-to-end workflow documented with diagram

## Modelling

- [x] Real target variable used or question reframed honestly
- [x] Split strategy justified
- [x] Class imbalance checked
- [x] Metrics reported correctly
- [x] Baseline and final model comparison included
- [x] Limitations stated honestly

## Code artefact zip

- [x] README with setup and run steps
- [x] requirements/environment file
- [x] scripts/notebooks included
- [x] data dictionary included
- [x] database creation / ingestion scripts included
- [x] output figures or instructions to regenerate included

## Presentation

- [ ] <= 10 minutes
- [ ] Full names + student numbers visible at start
- [ ] Problem, method, findings, impact all covered

## Journals

- [x] One PDF per member
- [x] Dates included
- [x] Tasks included
- [x] Time spent included
- [x] Challenges included
- [x] Resolutions included

## Final integrity check

- [x] No unsupported “first ever” claim
- [x] No dataset/model claim unsupported by code
- [x] No uncited borrowed text/figures/code
